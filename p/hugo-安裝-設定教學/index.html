安裝 Hugo

  • 新建資料夾 C:/hugo/bin
  • Hugo github release 下載對應壓縮檔 (推薦下載帶extended字尾的，以免後續有些主題套用上會有問題)
  • 把壓縮檔內容解壓縮到剛新建的資料夾內 (C:/hugo/bin)
  • 將資料夾路徑加到環境變數內，讓所有路徑都能使用 Hugo 指令
  • 檢查環境變數 & 確認 Hugo 是否安裝成功
1
2

echo %PATH%
hugo version

  • 複製 themes\hugo-theme-stack 下的 assets 資料夾到最外層
